# appleCounting

協助對應數字 → 數量，並展示減法動畫（數一數小蘋果）。

## 網址

- 倉庫：https://github.com/qsam23175401/appleCounting
- 主程式：倉庫中的 [`index.html`](./index.html)

## 基本用法

1. 用瀏覽器直接開啟 `index.html`，或在專案資料夾執行本機伺服器（例如 `npx http-server`）後開啟頁面。
2. 選擇模式：
   - **單蘋果模式**：只調整紅蘋果數量，練習數字與數量對應。
   - **雙蘋果模式**：同時調整紅、青蘋果數量。
3. 拉動拉桿改變數量，蘋果園會跟著更新。
4. **雙蘋果模式**可點「播放減法動畫」，用紅蘋果對應青蘋果做減法示範（紅蘋果數量需 ≤ 青蘋果）。
5. 點「隨機展示」可隨機產生一組數量快速練習。
